# Gilded Rose

### Setup


1. Clone the repo
    ```
    git clone git@github.com:matthewwoodruff/gilded-rose.git $GOPATH/src/github.com/River-Island/gilded-rose
    ```

2. Change directory
    ```
    cd $GOPATH/src/github.com/River-Island/gilded-rose
    ```
    
3. Run the tests
    ```
    go test
    ```
