# Creek and Continent

### Setup


1. Clone the repo
    ```
    git clone git@github.com:matthewwoodruff/creek-and-continent.git $GOPATH/src/github.com/River-Island/creek-and-continent
    ```

2. Change directory
    ```
    cd $GOPATH/src/github.com/River-Island/creek-and-continent
    ```
    
3. Run the tests
    ```
    go test
    ```
